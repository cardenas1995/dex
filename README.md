# Dex

## Overview
The Dex is a web application that allows users to browse and explore data for various Pokémon using the PokeAPI. Users can click on individual Pokémon to view their images, types, and descriptions.

## Features
- **Dynamic Data Fetching**: Loads data for up to 300 Pokémon from the PokeAPI.
- **Interactive UI**: Displays Pokémon names, images, and types, and updates descriptions upon selection.
- **Hover Effects**: Shows the back image of Pokémon when hovering over their name.
- **Responsive Design**: Adapts to various screen sizes for optimal user experience.

## Technologies Used
- **HTML**: For structuring the web pages.
- **CSS**: For styling the application and enhancing user interface.
- **JavaScript**: For dynamic content loading and interactivity.
- **Fetch API**: For making HTTP requests to the PokeAPI.
